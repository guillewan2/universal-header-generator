# Universal Header Generator
A tool to generate universal headers for programming projects. It supports multiple programming languages and can be customized to fit your project's needs.

It creates the file with the header at the top, ensuring consistency across your codebase.

## Languages Supported
- Shell scripts (Bash, Zsh, ...)
- Python
- PHP


## Use
1. Clone the repository:

2. Navigate to the project directory:

3. Run the script with the desired parameters:

```bash
./header-gen <filename> <The description you need for your project>
```
The result is something like this:
```bash
#!/usr/bin/python3

# Guillermo Torres
# Date: s√°b 06 sep 2025
# Time: 19:47
# Description: ''
# Distribution: Ubuntu 24.04.3 LTS
```
Make sure to replace `<filename>` with the name of your file and `<The description you need for your project>` with an appropriate description.
